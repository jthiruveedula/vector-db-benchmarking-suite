What we measure

Eight dimensions, one apples-to-apples harness

Every database runs the same workload generator, the same query set, and the same ground truth — so the numbers are comparable.

Recall@k

Latency

Throughput

Normalized $ per million queries across managed and self-hosted deployments.

Key findings

What the data actually says

1

Qdrant offers the best recall-to-latency tradeoff for self-hosted deployments.

2

pgvector is compelling for teams already on Postgres but falls short past ~10M vectors.

3

FAISS is unmatched for pure in-memory throughput but lacks persistence and filtering.

4

Pinecone serverless gives near-zero ops overhead at ~10x the cost of self-hosted options.

5

Hybrid search (BM25 + dense) in Weaviate lifts recall@10 by 8-12% on keyword-heavy queries.

6

Highly selective metadata filters degrade most ANN indexes severely — payload-indexed filtering wins.
